AI-powered cryptocurrency analysis platform with hidden gem detection and weekly email reports.

## Quick Start

```bash
pip install -e .
python app.py
# Visit http://localhost:5001
```

## Weekly Email Setup

```bash
export REPORT_EMAIL_FROM="your-email@gmail.com"
export REPORT_EMAIL_TO="your-email@gmail.com"
export SMTP_PASSWORD="your-gmail-app-password"
```

See [GUIDE.md](GUIDE.md) for full setup instructions.

## Features

- Real-time crypto analysis with ML predictions
- Hidden gem detection AI
- Weekly email reports (Monday 9 AM)
- On-demand learning (cost-effective)
